void drive_forward ( )

Function for driving the car forward.

void initialize_car_drive ( )

Function for initializing necessary things for driving.

This function initializes I2C interface that is needed for configuring the I2C driver on the vehicles driver board. It also then configures I2C driver in way that driving of the vehicle is enabled.
