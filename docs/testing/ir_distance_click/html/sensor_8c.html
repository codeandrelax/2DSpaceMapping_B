Detailed Description

Testing of IR Distance Click sensor.

This file contains code that is used as tester for sensor used in the project. It takes sensor reading as analog values and using AD convertor converts it into some digital values that is then stored into some variable. It is then transformed into float value that represesents voltage that sensor generates, according to sensor datasheet. When some value is got (1.5 for about 10 cm ) LED diode connected to some pin is ON.
