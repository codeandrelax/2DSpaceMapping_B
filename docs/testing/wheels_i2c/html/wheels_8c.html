Detailed Description

File containing testing program for driving the car.

This file is used for initializing communication with driver board of the system, or precisely, with the I2C driver on the board namely PCA9633 component. Several I2C configuration messages are needed for component to initialize, in which some configuration register are filled. After that I2C messages that set register for driving can be sent.
